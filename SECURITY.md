# Security Policy

## Supported Versions

Security fixes are handled on the latest `main` branch and the most recent npm
package version.

## Reporting a Vulnerability

Please report suspected vulnerabilities by opening a private GitHub security
advisory for this repository. Include:

- the affected version or commit
- a minimal reproduction or input file
- expected and observed behavior
- any known workaround

Do not include sensitive customer data, private source bundles, or unpublished
drafts in public issues.

## Scope

This package is a local CLI and library. It reads local draft/source files and
prints reports; it should not publish content or contact external services as
part of normal operation.
