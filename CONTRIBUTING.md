# Contributing

Thanks for helping improve `agent-claim-check-skill`.

## Development

```bash
npm install
npm run release:check
```

Use synthetic drafts and source bundles in tests. Do not commit private documents, customer data, credentials, or proprietary source text.

## Pull Requests

- Describe the claim-classification behavior being changed.
- Add or update fixture-backed tests when scoring or report output changes.
- Run `npm run release:check` before requesting review.
